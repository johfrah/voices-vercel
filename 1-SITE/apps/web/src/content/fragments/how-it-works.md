# Hoe het werkt

1. **Kies jouw stem**
Filter op taal, geslacht en stijl in onze database.

2. **Briefing & Script**
Upload jouw script of geef jouw briefing door via de chat.

3. **Pro Studio Opname**
Onze stemacteurs nemen jouw tekst op in topkwaliteit.

4. **Directe Levering**
Ontvang jouw bestanden binnen 24 uur. Klaar voor gebruik.
